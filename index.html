Dashboard Interactivo
Descripción del Proyecto

Desarrollo de un dashboard a medida para visualizar métricas clave y el estado general del negocio. Incluye funcionalidades como registro dinámico en agenda, gráficos interactivos (implementados con Chart.js), opción de exportación de datos relevantes, un calendario de pagos integrado, y una página comercial para promoción. Se implementaron controles para evitar la superposición de horarios y cálculo automático del monto para clases individuales. Optimizado para un rendimiento eficiente incluso con grandes volúmenes de información.

Tecnologías Empleadas

ASP.NET Razor Pages SQL Server C# JavaScript (ES6+) Bootstrap 5 Chart.js Git & GitHub
